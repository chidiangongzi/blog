personal ideal
改写的login.sh登录学校新堡垒大型机的脚本文件重命名为login，放置于/usr/bin/下可以随时login+六位数OTP登录学校新大型机。
改写的nuist.py需要在开头加上#!/bin/python 并将nuist重命名为nuist，防止于/usr/bin下可随时nuist链接校内以太网。
