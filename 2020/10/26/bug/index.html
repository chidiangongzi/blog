修复Archlinux上的GPGME error（when trying to upgrade with pacman -Syyu

1.删除旧的同步的db文件

sudo rm /var/lib/pacman/sync/*

2.重新生成pacman-key

sudo pacman-key --init

3.重新Populate pacman-key

sudo pacman-key --populate

4.重新同步

sudo pacman -Syyy

安装了conda后，每行最上面会有一个(base)，很烦人，终于找到最佳解决方案了：

conda config --set auto_activate_base false

原因：
安装conda后，每次启动终端，都会自动启动conda的base环境，conda的环境可以用 conda env list 查看
只要设置conda不要自动启动base环境就可以了
