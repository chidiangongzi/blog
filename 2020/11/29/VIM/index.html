删除行尾空格

命令1：%s/\s\+$//g.可以删除行尾多余的空格和tab符号。

命令2：%s/ *$//g. /和*中间有空格，只能删除行尾空格，对于tab不能删除

删除某行前n个字符

:%s/^.{n}//

把某几行所有空格全部删除。

:’<,’>s/\s+//g

vim基础操作

commond模式按’`‘可以将大写改成小写
制表符插入模式下ctrl v +Tab可获得制表符号

vim添加未识别文件类型的解决方法

vim用户

在ncl.vim内添加如下内容

au BufRead,BufNewFilE *.ncl set filetype=ncl

neovim用户

在ncl.vim内添加如下内容

autocmd BufNewFile,BufRead *.ncl set filetype=ncl

vim宏操作

  1. 基本使用

    q[a-z] 开始录制宏

    q 停止录制

    @[a-z] 使用宏

    @@ 调用最近使用的宏

    22@[a-z] 多次重放宏

  2. 宏的执行方式

串行方式：5@[a-z] 宏内包含向下一个目标行进行移动的指令，如果出现错误，宏会停止执行

并行方式：:normal @[a-z] 在高亮区域的每一行上并行执行宏，某一行出现错误不影响其他行

vim 的coc (Conquer of Completion) 模块

  • Make your Vim/Neovim as smart as VSCode

  • 可以用Javascript或typescript写vim插件

  • 补全协议 language server protocol

安装依赖

Install nodejs>=10.12

curl -sL install-node.now.sh/lts | bash或brew install node

sudo npm i -g neovim yarn

使用Vim-plug安装coc.nvim


检验是否健康(仅Neovim可用):checkheath

安装coc扩展 (给coc安装插件)

Install extensions like :CocInstall coc-json

Uninstall extensions like :CocUninstall coc-json

查看已经安装的插件:CocList extensions

自动批量安装插件在init.vim或.vimrc中加入

如何查找发现插件

安装coc-marketplace :CocInstall coc-marketplace

加载coc扩展 :CocList marketplace

TextEdit might fail if hidden is not set vim未保存的文件还在缓冲区
set hidden
推荐设置成100，使vim响应更快
set updatetime=100

使得补全信息栏少出现一些没用的
set shortmess+=c

使按下Tab可以自动补全代码

手动调出可以补全的内容，默认ctrl+空格inoremap <silent><expr> <c-space> coc#refresh()和个人快捷键调用输入法冲突，改为inoremap <silent><expr> <c-o> coc#refresh()
