xinput set-prop 10 "Device Enabled" 1

linux下如何查看CPU信息：

cat /proc/cpuinfo

model name cpu名称，型号，主频
cpu Mhz 实际主频
physical id 单个cpu的标号
cpu cores 逻辑核所处cpu的物理核
siblings 单个cpu的逻辑核数

查看cpu有多少个核

cat /proc/cpuinfo | grep "physical id" | sort | uniq | wc -l

查看单个cpu的物理核数

cat /proc/cpuinfo | grep "cpu cores" | uniq

查看总逻辑核数

cat /proc/cpuinfo | grep "processor" | wc -l

查看所有逻辑核的负载

执行top命令，然后按1

linux下wc用法小节

统计行数

wc -l file

cat file | wc -l(只返回数字)

统计单词数

wc -w file

cat file | wc -w

统计字符数

wc -c file

cat file | wc -c

echo -n 1234 | wc -c(-n用于避免echo添加额外的换行符)

结果返回4

不执行任何选项执行wc时

wc file

结果打印出文件的行数、单词数、字符数

打印出文件中最长一行的长度

wc file -L
