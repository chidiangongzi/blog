wrf_interp

WRF在垂直方向的插值

官网下载并安装wrf_interp https://www2.mmm.ucar.edu/wrf/users/download/get_sources_pproc_util.html

解压缩：tar -xvzf WRF_INTERP.TAR.gz

编译成可执行文件ifort wrf_interp.F90 -L$NETCDF/lib -lnetcdff -lm -I$NETCDF/include -free -o interp.exe

cdo selvar,U wrfout_d02_2017-06-07_00\:00\:00 U.nc

cdo seltimestep,182 wrfout_d02_2017-06-07_00\:00\:00 182time.nc

ARWpost 的安装注意

NCO工具使用

ncks -A -v QRAIN wrfout_d02_2017-06-07_00\:00\:00 rain.nc

ncks -d Time,2,3 rain.nc rain-2-3.nc

-d 维度名，-v变量名，重复的维度名会报错
ncrename -d x,west_east -d x_2,bottom_top -d x_3,bottom_top_stag -d x_4,soil_layers_stag -d x_5,west_east_stag -d x_6,west_east -d y,south_north -d y_2,south_north -d y_3,south_north_stag -d lev,bottom_top -d lev_2,bottom_top_stag -d lev_3,soil_layers_stag 181time.nc 181time-rename.nc

ncpdq –rdr=Time,DateStrLen,west_east,south_north,bottom_top,bottom_top_stag,soil_layers_stag,west_east_stag,south_north_stag wrfout_d02_73_00\:00\:00 order.nc

ncpdq –rdr=-Time,-DateStrLen,-west_east,-south_north,-bottom_top,-bottom_top_stag,-soil_layers_stag,-west_east_stag,-south_north_stag wrfout_d02_73_00\:00\:00 order.nc
