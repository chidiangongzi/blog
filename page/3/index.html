窗口管理器dwm使用过程中的一些快捷键

key Command
$mod+Enter 启动虚拟终端，将新打开的设为主视角
$mod+P 熄屏
$mod+F 全屏
$mod+k 隐藏
$mod+K 恢复隐藏
$mod+t Title模式
$mod+m 全铺模式
$mod+空格 全铺和Title模式中切换
$mod+I 将当前窗口移动到右边标签
$mod+N 将当前窗口移动到左标签
$mod+h 调整窗口大小向左
$mod+l 调整窗口大小向右
$mod+H 调整水平或垂直分割
$mod+L 调整水平或垂直分割
